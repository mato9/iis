Instalace

  • Instalace probíha překopírovaním souborů na server do cílové složky a změnou souboru app/application/config/database.php na správné hodnoty pro připojení k databázi.
  • Stránky používají framework CodeIgniter a základní konfigurace je postačující - PHP 5.4 a MySQL 5.6.
  • Databáze se inicializuje skriptem create_tables_script.sql.
